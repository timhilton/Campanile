# Campanile

A lightweight macOS menu bar app that chimes the quarter hour, every hour.

## Features

- Rings at :15, :30, :45, and strikes the hour
- Adjustable volume
- Mute with one click
- Quiet Hours: set a custom schedule to silence the bells automatically
- Starts at login

## Requirements

- macOS 13.0 or later

## Building

Open `Campanile.xcodeproj` in Xcode and build the target.
